3) Дан массив из большого числа элементов (числа), отсортированный по возрастанию. Необходимо написать функцию, 
которая найдет индекс элемента массива, значение по которому наиболее близко к переданному в аргументах функции числу. 
Код необходимо оформить в виде класса (т.е. должен быть метод new возвращающий экземпляр) с именем 
ИмяФамилияЛатиницейFindIndex.pm, у объекта класса должен быть метод find принимающий на вход искомое 
значение и ссылку на массив, метод должен возвращать массив из двух элементов - найденных индекс и 
число шагов (число сравнений).
